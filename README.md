## Описание
Это приложение представляет собой REST API для управления балансом кошелька. 
Оно поддерживает операции по зачислению (DEPOSIT) и снятию (WITHDRAW) средств, а также позволяет получать текущий баланс кошелька.

## Технологический стек
- Java (версии 8-17)
- Spring Boot 3
- PostgreSQL
- Liquibase для миграции базы данных
- Docker и Docker Compose для контейнеризации
- JUnit и Mockito для тестирования

## Требования
- Docker
- Docker Compose
